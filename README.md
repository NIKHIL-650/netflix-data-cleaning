# netflix-data-cleaning
Data cleaning and preprocessing of Netflix Movies and TV Shows dataset using Excel
# Netflix Data Cleaning Project

## Dataset
Netflix Movies and TV Shows dataset from Kaggle

## Tools Used
- Microsoft Excel

## Tasks Performed
- Removed duplicates using show_id
- Handled missing values (director, cast, country)
- Cleaned text using TRIM and PROPER
- Converted date_added to proper date format
- Separated raw and cleaned data

## Files
- Cleaned_dataset.xlsx (raw + cleaned sheets)
- cleaned_dataset.csv (final cleaned data)

## Outcome
Produced a clean and structured dataset ready for analysis.
